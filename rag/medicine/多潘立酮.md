# 多潘立酮

- 别名：多潘立酮片、吗丁啉
- 规格：常见 10mg
- 类别：促胃肠动力药
- 治疗用途：用于消化不良、饱胀、上腹不适、嗳气、恶心、呕吐等，通过促进胃肠蠕动帮助胃排空。
- 用法用量：一般在餐前 15~30 分钟服用，成人常见一次 10mg、一日 3 次；儿童及老年人须遵医嘱减量。不建议自行长期服用。
- 注意事项：
  1. 严重心脏病、心律失常、明显电解质紊乱者及正在服用可延长 QT 间期药物者应避免使用，用前咨询医生。
  2. 不宜与红霉素、酮康唑等药物同用，可能增加心脏风险。
  3. 出现心悸、晕厥等症状应立即停药就医。
  4. 症状持续不缓解或反复呕吐、体重下降，应就医查明原因。
