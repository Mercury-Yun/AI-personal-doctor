# 碘伏

- 别名：聚维酮碘溶液、碘伏消毒液
- 规格：常见 0.5%、1% 溶液
- 类别：外用消毒防腐
- 治疗用途：用于皮肤、小伤口、擦伤及手术前皮肤的消毒，也可用于黏膜的消毒，杀菌温和、刺激性较小，是家庭常备外用消毒剂。
- 用法用量：外用。用棉签或棉球蘸取适量，由伤口中心向外涂擦消毒；一般无须再用酒精脱碘。仅供外用，不可口服。
- 注意事项：
  1. 对碘过敏者、甲状腺功能异常者慎用；大面积、长期使用前咨询医生。
  2. 深而大的伤口、动物咬伤、伤口有异物或红肿化脓时，应就医处理，必要时接种破伤风疫苗。
  3. 避免接触眼睛；开封后注意密封避光保存，变色或过期不再使用。
  4. 婴幼儿、孕妇大面积使用需谨慎，遵医嘱。
