# 蒙脱石散

- 别名：蒙脱石散剂、思密达
- 规格：常见 3g/袋
- 类别：止泻（肠道黏膜保护剂）
- 治疗用途：用于成人及儿童急、慢性腹泻，通过覆盖和保护肠道黏膜、吸附病原和毒素来减轻腹泻，对水样便效果较好。
- 用法用量：需用温水冲成混悬液后服用。成人一般一次 1 袋（3g）、一日 3 次；儿童按年龄减量，具体遵医嘱或参照说明书。空腹服用吸附效果更好。
- 注意事项：
  1. 与其他药物间隔至少 1~2 小时服用，以免吸附影响其他药物的吸收。
  2. 腹泻期间注意补充水分和电解质（可配合口服补液盐），防止脱水。
  3. 若腹泻伴高热、明显血便、持续呕吐或超过 2~3 天不缓解，应及时就医。
  4. 长期便秘者慎用；婴幼儿使用应在医生指导下进行。
