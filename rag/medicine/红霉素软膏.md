# 红霉素软膏

- 别名：红霉素眼膏（不同剂型）、红霉素软膏
- 规格：常见 1%（软膏）
- 类别：外用抗菌药
- 治疗用途：用于脓疱疮等化脓性皮肤病，以及小面积皮肤感染、轻微擦伤后的预防感染，也常用于鼻腔、口角轻度炎症的护理。
- 用法用量：外用，取适量涂于患处，一日 1~2 次，涂薄薄一层即可。眼部感染需使用专门的红霉素眼膏，不可混用。
- 注意事项：
  1. 对红霉素或大环内酯类抗生素过敏者禁用。
  2. 仅供外用，避免长期、大面积使用以减少耐药和刺激。
  3. 用药部位若出现红肿、瘙痒等过敏反应应停药。
  4. 感染范围扩大、发热或伤口化脓明显，应及时就医，不要单靠外用药。
