# 二甲双胍

- 别名：盐酸二甲双胍片、盐酸二甲双胍缓释片、格华止
- 规格：常见 0.5g、0.85g
- 类别：口服降糖药（双胍类）
- 治疗用途：2 型糖尿病常用的基础降糖药，通过减少肝糖输出、改善胰岛素抵抗来降低血糖，单独使用一般不引起低血糖。
- 用法用量：建议随餐或餐后服用以减少胃肠不适；缓释片须整片吞服。通常从小剂量起始逐步加量，具体剂量遵医嘱。
- 注意事项：
  1. 肾功能明显下降、严重感染、脱水、缺氧或进行含碘造影检查前后，应按医生要求暂停或调整。
  2. 避免大量饮酒，以免增加乳酸酸中毒风险。
  3. 服药早期可能出现胃部不适、腹胀、恶心、腹泻，多数可逐渐适应。
  4. 规律监测血糖、糖化血红蛋白，定期检查肾功能；饮食控制与运动同样重要。
