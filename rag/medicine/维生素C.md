# 维生素C

- 别名：维生素C片、维C、抗坏血酸
- 规格：常见 100mg（片）
- 类别：维生素补充剂
- 治疗用途：用于预防和治疗维生素 C 缺乏（坏血病），也用于病后恢复期、慢性消耗性疾病等的辅助补充，参与机体多种代谢。
- 用法用量：口服，作为补充预防成人常见一次 100mg 左右、一日 1~3 次；治疗剂量遵医嘱。饭后服用可减少胃部刺激。
- 注意事项：
  1. 不宜长期大剂量服用，过量可能增加泌尿系结石风险、引起胃肠不适。
  2. 肾结石病史、痛风、糖尿病患者大剂量使用前咨询医生。
  3. 均衡饮食（新鲜蔬果）通常已能满足日常需要，不必盲目大量补充。
  4. 服用大剂量后突然停药需逐渐减量，避免出现反跳性缺乏。
