# 蜜炼川贝枇杷膏

- 别名：川贝枇杷膏、京都念慈菴蜜炼川贝枇杷膏
- 规格：常见瓶装糖浆/膏剂（川贝母、枇杷叶、蜂蜜等）
- 类别：润肺止咳化痰类中成药
- 治疗用途：用于肺燥、痰热引起的咳嗽、痰稠、咽干喉痒、声音嘶哑等。
- 用法用量：口服，成人一次约 15~22ml（约 1~2 汤匙）、一日 3 次，可含咽慢服；儿童减半。可用温水调服。
- 注意事项：
  1. 含糖量高，糖尿病患者慎用或遵医嘱；风寒咳嗽（痰白清稀、怕冷）者不宜。
  2. 属对症止咳，不宜长期大量服用；有痰应同时注意排痰。
  3. 服药期间忌食辛辣、油腻、鱼腥食物。
  4. 咳嗽持续 1 周以上、伴发热、气促或痰中带血，应就医查因。
