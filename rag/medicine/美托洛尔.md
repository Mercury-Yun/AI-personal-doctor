# 美托洛尔

- 别名：酒石酸美托洛尔片、琥珀酸美托洛尔缓释片、倍他乐克
- 规格：常见 25mg、47.5mg（缓释）
- 类别：降压/抗心律失常（β受体阻滞剂，处方药）
- 治疗用途：用于高血压、心绞痛、部分心律失常及慢性心力衰竭等，通过减慢心率、降低心肌耗氧来保护心脏。属处方药，须遵医嘱服用。
- 用法用量：口服，缓释片须整片吞服、不可掰碎；剂量由医生根据心率和血压逐步调整，通常固定每天时间服用。不可自行突然停药。
- 注意事项：
  1. 突然停药可能诱发血压反跳、心绞痛加重甚至心梗，需在医生指导下逐渐减量。
  2. 明显心动过缓、严重房室传导阻滞、失代偿心衰、哮喘急性发作者禁用或慎用。
  3. 糖尿病患者用药可能掩盖低血糖的心悸表现，需加强血糖监测。
  4. 用药期间监测心率和血压，出现明显乏力、气促、心率过慢应就医。
