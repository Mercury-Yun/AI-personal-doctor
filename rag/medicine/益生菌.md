# 益生菌

- 别名：双歧杆菌活菌、地衣芽孢杆菌（整肠生）、布拉氏酵母菌、酪酸梭菌等微生态制剂
- 规格：胶囊、片剂、散剂等多种剂型
- 类别：肠道微生态调节剂
- 治疗用途：用于肠道菌群失调引起的腹泻、便秘、腹胀，也用于抗生素相关性腹泻、消化不良的辅助调理。
- 用法用量：按说明书或医嘱，一般一日 2~3 次；活菌制剂宜用温水（不超过 40℃）送服，避免开水冲泡杀死活菌。
- 注意事项：
  1. 多数活菌制剂需与抗生素间隔 2~3 小时服用，避免抗生素杀灭活菌；或选耐药型菌株。
  2. 活菌产品需按说明书低温或阴凉保存，避免高温；开封后尽快服完。
  3. 免疫功能严重低下、危重或有中心静脉置管者用前咨询医生。
  4. 腹泻伴高热、血便、剧烈腹痛或持续不缓解，应及时就医，不要仅靠益生菌。
