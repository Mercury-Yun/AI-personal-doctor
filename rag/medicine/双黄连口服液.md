# 双黄连口服液

- 别名：双黄连口服液、双黄连合剂（金银花、黄芩、连翘）
- 规格：常见每支 10ml
- 类别：清热解毒类中成药
- 治疗用途：用于外感风热所致的感冒，症见发热、咳嗽、咽痛等，多作对症辅助。
- 用法用量：口服，成人一次 1~2 支（10~20ml）、一日 3 次，或遵说明书；儿童减量。摇匀后服用，症状缓解即停。
- 注意事项：
  1. 性偏寒凉，风寒感冒（怕冷、无汗、清涕）者不宜；脾胃虚寒、易腹泻者慎用。
  2. 糖尿病患者及孕妇慎用；不宜与滋补性中药同用。
  3. 对本品过敏者禁用，服后出现皮疹、腹泻应停药；不建议无症状时长期服用预防。
  4. 服用 3 天症状不缓解、或高热咽痛加重，应就医。
