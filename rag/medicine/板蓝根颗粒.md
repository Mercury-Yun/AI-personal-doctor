# 板蓝根颗粒

- 别名：板蓝根冲剂、板蓝根颗粒（含无糖型）
- 规格：常见每袋 5g、10g
- 类别：清热解毒类中成药
- 治疗用途：用于风热感冒、咽喉肿痛，以及病毒性感冒、咽炎的辅助对症调理。
- 用法用量：开水冲服，成人一次 1~2 袋、一日 3~4 次，或遵说明书；儿童减量。症状缓解即停，不宜长期当保健品服用。
- 注意事项：
  1. 属清热解毒药，性偏寒凉，风寒感冒（怕冷、清涕、无咽痛）者不宜；脾胃虚寒、易腹泻者慎用。
  2. 糖尿病患者选无糖型；不宜与滋补性中药同服。
  3. 没有确切预防感冒作用，不建议无症状时长期服用来"防病"。
  4. 服用 3 天症状无改善、或高热、咽痛剧烈、化脓，应就医。
