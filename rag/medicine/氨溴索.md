# 氨溴索

- 别名：盐酸氨溴索片、盐酸氨溴索口服液、沐舒坦
- 规格：常见 30mg（片）
- 类别：祛痰（黏液溶解剂）
- 治疗用途：用于痰液黏稠、不易咳出的急慢性呼吸道疾病，能稀释痰液、促进排痰，常用于支气管炎、肺炎等的辅助祛痰。
- 用法用量：成人常见一次 30mg、一日 3 次，餐后服用；儿童按年龄和体重减量，遵医嘱或参照说明书。服药期间多喝水有助排痰。
- 注意事项：
  1. 一般不与强力镇咳药同时使用，以免痰液稀释后不能咳出而滞留。
  2. 胃溃疡患者、孕早期妇女慎用，用前咨询医生。
  3. 偶见胃部不适、恶心、皮疹，出现明显过敏应停药。
  4. 咳嗽咳痰持续加重、伴高热或呼吸困难，应就医查明病因。
