# 阿托伐他汀

- 别名：阿托伐他汀钙片、立普妥
- 规格：常见 10mg、20mg
- 类别：调脂药（他汀类，处方药）
- 治疗用途：用于降低血液中的低密度脂蛋白胆固醇（“坏胆固醇”），防治动脉粥样硬化，降低冠心病、脑卒中等心血管事件风险。属处方药，须遵医嘱长期规律服用。
- 用法用量：口服，一日 1 次，可在一天中任意固定时间服用（不受进餐限制），常见睡前或晚上服；起始剂量和调整由医生根据血脂和病情决定，不可自行增减或停药。
- 注意事项：
  1. 用药期间如出现不明原因的肌肉疼痛、无力、酱油色尿，应立即停药就医（警惕肌病）。
  2. 需定期复查血脂、肝功能和肌酶；活动性肝病者禁用。
  3. 避免与大量西柚汁同服；合并其他降脂药或某些抗生素前告知医生。
  4. 孕妇、哺乳期禁用；调脂同时坚持低脂饮食、戒烟限酒和运动。
