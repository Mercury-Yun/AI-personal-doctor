# 氯苯那敏

- 别名：马来酸氯苯那敏、扑尔敏
- 规格：常见 4mg（片）
- 类别：第一代抗组胺（抗过敏）药
- 治疗用途：用于皮肤过敏（荨麻疹、湿疹、皮肤瘙痒）、过敏性鼻炎，也常作为复方感冒药成分缓解打喷嚏、流涕。
- 用法用量：成人一次 4mg（1 片）、一日 3 次，或遵医嘱；儿童及老人减量。症状缓解即停，不宜长期服用。
- 注意事项：
  1. 明显嗜睡副作用，服后不要驾车、操作机械、高空作业；老人服后需防跌倒。
  2. 青光眼、前列腺肥大、幽门梗阻者慎用；许多复方感冒药已含本成分，避免叠加过量。
  3. 服药期间忌酒及镇静类药物，以免嗜睡加重。
  4. 过敏症状严重（呼吸困难、面部肿胀）应立即就医，不能只靠口服抗过敏药。
