# 藿香正气水

- 别名：藿香正气水、藿香正气口服液、藿香正气软胶囊
- 规格：常见每支 10ml（水剂）
- 类别：中成药（解表化湿、理气和中）
- 治疗用途：用于外感风寒、内伤湿滞或夏季暑湿引起的头痛昏重、胸闷、恶心呕吐、腹泻等，常用于胃肠型感冒和中暑不适。
- 用法用量：口服，成人常见一次 5~10ml（一支左右）、一日 2 次，服前摇匀；具体参照说明书，儿童遵医嘱减量。
- 注意事项：
  1. 水剂含有乙醇（酒精），服后不要驾车、操作机械；正在服用头孢类等药物者禁用水剂，以免双硫仑样反应，可改用不含酒精的软胶囊或口服液。
  2. 对酒精过敏者、孕妇慎用或遵医嘱。
  3. 不宜与滋补性中药同服；服药期间忌生冷油腻。
  4. 症状持续不缓解、腹泻严重或高热，应就医。
