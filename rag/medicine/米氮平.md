# 米氮平

- 别名：米氮平片、瑞美隆
- 规格：常见 15mg、30mg
- 类别：抗抑郁药（处方药）
- 治疗用途：用于抑郁症，也用于部分焦虑相关症状及失眠（尤其早醒型）的辅助治疗。属处方药，须遵医嘱服用，不可自行增减或突然停药。
- 用法用量：常见睡前口服，具体剂量由医生根据病情调整；起效通常需要数天至数周。不要因短期感觉好转自行停药。
- 注意事项：
  1. 初期可能出现嗜睡、口干、食欲增加，服药期间避免饮酒。
  2. 若出现明显镇静过度、情绪恶化或自杀念头，应立即联系医生。
  3. 老年人、肝功能异常者需减量并密切随访。
  4. 与某些药物存在相互作用，合并用药前咨询医生或药师；复诊时带上药盒便于核对（不同厂家商品名可能不同）。
