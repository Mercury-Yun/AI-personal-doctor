# 奥美拉唑

- 别名：奥美拉唑肠溶胶囊、奥美拉唑肠溶片、洛赛克
- 规格：常见 20mg
- 类别：抑酸护胃（质子泵抑制剂 PPI）
- 治疗用途：减少胃酸分泌，用于胃食管反流、胃溃疡、十二指肠溃疡，以及部分服用抗血小板药患者的胃黏膜保护，缓解反酸、烧心、胃痛。
- 用法用量：通常早餐前空腹口服，肠溶剂型须整粒吞服、不要嚼碎；具体剂量和疗程遵医嘱，不建议长期自行服用。
- 注意事项：
  1. 长期使用可能影响维生素 B12、镁、钙的吸收，并可能增加肠道感染风险。
  2. 与部分药物（如氯吡格雷等）存在相互作用，合并用药前咨询医生或药师。
  3. 若伴消瘦、吞咽困难、黑便或持续呕吐，应及时就医排查。
  4. 配合少吃辛辣油腻、避免过晚进食；症状反复应评估幽门螺杆菌和胃镜检查需要。
