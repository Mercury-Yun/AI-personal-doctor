# 阿莫西林

- 别名：阿莫西林胶囊、阿莫西林片、羟氨苄青霉素
- 规格：常见 0.25g、0.5g
- 类别：抗生素（青霉素类，处方药）
- 治疗用途：用于敏感细菌引起的感染，如某些呼吸道、泌尿道、皮肤软组织感染等，也用于幽门螺杆菌根除方案。属处方药，须经医生判断细菌感染后使用。
- 用法用量：口服，成人常见一次 0.5g、每 6~8 小时一次，具体剂量、疗程由医生决定；须按疗程足量服用，不可症状好转就自行停药，以免耐药或复发。
- 注意事项：
  1. 对青霉素类过敏者禁用；用药前务必告知医生过敏史，出现皮疹、呼吸困难等过敏反应立即停药就医。
  2. 病毒性感冒、流感等非细菌感染不应使用抗生素。
  3. 可能引起腹泻、恶心、皮疹；服药期间避免饮酒。
  4. 严格遵医嘱按时按量、完成整个疗程，不要与他人共用或自行留用剩药。
